The Cyber Kill Chain

The Cyber Kill Chain is a model used to describe the stages of a cyberattack, from initial reconnaissance to the final exfiltration of data. The stages typically include:

Stage Name Description
1 Reconnaissance Gathering information about the target, such as IP addresses, employee names, and technology stack.
2 Weaponization Creating a malicious payload (e.g., malware embedded in a document or script).
3 Delivery Sending the malicious payload to the target via email, web, USB, or another method.
4 Exploitation Executing the malicious code by exploiting a vulnerability or user action.
5 Installation Installing malware on the target system to maintain access.
6 Command and Control (C2) Establishing a remote connection to control the infected system.
7 Actions on Objectives Executing the final goal of the attack (e.g., data theft, destruction, espionage).
