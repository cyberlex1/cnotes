Open Source or Close Source
Open Source can be more secure if actively maintained and has a strong community that continuously reviews the code. Closed Source can be more secure if it's a well-established, professionally maintained product with strong security protocols.

Preventive Controls
Measures put in place to prevent security incidents from occurring. Examples: Firewalls, Encryption, Access Controls, MFA. Goal: block/deter malicious activities before they happen.
Detective Controls
Measures that detect and identify security incidents. Examples: IDS, Log Monitoring, Cameras, File Integrity Monitoring. Goal: identify/respond to attacks that bypass prevention.
