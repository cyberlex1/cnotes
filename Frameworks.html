XSS

Parameterized queries are SQL queries where user inputs are passed as parameters rather than being directly embedded into the query string. This prevents SQL injection by treating user input as data, not executable code. It ensures the query structure remains safe and intact.

Best Practices / Methodologies

Framework/List Description Example
OWASP An open-source organization focused on improving web application security. The OWASP Top 10 is a list of the most critical security risks for web applications. Injection (e.g., SQL Injection)
PTES Penetration Testing Execution Standard provides guidelines for conducting penetration tests, focusing on key steps such as information gathering, vulnerability analysis, and exploitation. Information Gathering (collecting data about the network)
SANS 25 A list of the top 25 software errors that result in security vulnerabilities, highlighting common coding mistakes that lead to vulnerabilities. Buffer Overflow (excessive data written to a buffer)

Cybersecurity Frameworks

Framework/List Description Example
COBIT Framework for IT governance and management, focusing on aligning IT with business objectives and ensuring effective governance. Control Objectives for IT (COBIT 5)
NIST Provides guidelines and best practices for improving cybersecurity, including the NIST Cybersecurity Framework (CSF). Identify, Protect, Detect, Respond, Recover
MITRE ATT&CK A framework that describes the tactics, techniques, and procedures (TTPs) used by adversaries during cyber attacks. It helps security professionals identify and defend against specific threats. Spearphishing Attachment (sending a malicious attachment via email)

Standards and Regulations

Standard/Regulation Description Example
PCI DSS Payment Card Industry Data Security Standard: a set of security standards to ensure that companies handling credit card information maintain secure systems and processes. Encryption of cardholder data, maintaining secure networks
ISO/IEC 27001 International standard for Information Security Management Systems (ISMS), providing a systematic approach to managing sensitive company information. Establishing and maintaining a risk management process for protecting information assets
HIPAA Health Insurance Portability and Accountability Act: U.S. regulation focused on protecting the privacy and security of healthcare information. Data encryption, secure communication of patient information
GDPR General Data Protection Regulation: EU regulation focused on data protection and privacy for all individuals within the European Union. Right to be forgotten, data subject consent for processing personal data
