What does LDAP allow?

LDAP (Lightweight Directory Access Protocol) is used for accessing and querying directory services like Active Directory.

  • Search for users, groups, computers, printers, etc.
  • Authenticate users (via bind operation).
  • Retrieve attributes (e.g., email, phone number, group membership).
  • Organize data hierarchically (e.g., departments, locations).
  • Integrate applications with centralized user directories.

Note: LDAP is not encrypted by default. For secure communication, use LDAPS (LDAP over SSL/TLS).
