APT (Advanced Persistent Threat)

An APT is a prolonged, targeted cyberattack aiming to steal data or disrupt systems by maintaining stealthy, long-term access.

Persistent Threat Attacks: Descriptions and Examples

Persistent Threat Attack Description Example
APT (Advanced Persistent Threat) A long-term, targeted cyberattack in which attackers infiltrate a network and remain undetected to steal data or cause damage. The Stuxnet worm, which targeted industrial control systems over a prolonged period.
Backdoor Malicious software that allows attackers to bypass standard authentication and security measures for continued access. Backdoors installed by malware to maintain unauthorized access after an attack has been carried out.
Botnet Infected machines controlled remotely by a hacker, often used for ongoing DDoS (Distributed Denial of Service) attacks or data theft. Botnets used to flood networks with traffic or exfiltrate data over a period of time.
Credential Stuffing Using stolen credentials from one breach to try and access multiple other accounts over an extended period. Hackers use a database of email/password pairs to access a range of services like banking or social media accounts.
Exfiltration of Data Persistent efforts to steal sensitive data from a network, usually over an extended period, without being detected. Ongoing transfers of company intellectual property or customer data from compromised systems to an external server.
Phishing Campaigns Persistent attacks targeting individuals or organizations to steal sensitive information by mimicking trusted entities. Multiple phishing emails sent over weeks or months to collect login credentials from employees.
Rootkit A set of tools that allow an attacker to maintain privileged access to a system while hiding their presence. Installations of remote access tools (RATs) to maintain continuous control over a compromised system.
Spyware Malicious software designed to collect information from a target device without their knowledge. Keyloggers that record sensitive information over time, such as usernames and passwords.

Stages of an APT (Advanced Persistent Threat)

Stage Description
1. Reconnaissance The attacker gathers information about the target, such as vulnerabilities and weaknesses.
2. Initial Compromise The attacker gains access to the network through phishing, exploiting vulnerabilities, etc.
3. Establishing Foothold Install backdoors or malware to ensure long-term access to the system.
4. Lateral Movement Move across the network to find valuable data or systems.
5. Privilege Escalation Gain higher access privileges to increase control over the system.
6. Data Exfiltration Steal valuable data such as intellectual property or sensitive information.
7. Cleanup and Covering Tracks Remove traces of the attack and avoid detection by security systems.
8. Maintaining Access Ensure continued access to the network for future use or attacks.

Common Malware Attacks: Descriptions and Examples

Malware Attack Description Example
Virus Malicious software that attaches itself to clean files and spreads to other files and systems. Infected email attachment that spreads once opened.
Worm A self-replicating malware that spreads across networks without needing a host file. Code Red Worm that spread through vulnerable servers.
Trojan Horse A type of malware disguised as a legitimate program, often used to steal data or control a system. Fake software installer that gives hackers access to your computer.
Ransomware Malware that encrypts files and demands payment for decryption. WannaCry that demanded Bitcoin for file decryption.
Spyware Software that secretly monitors and collects user data, often for advertising purposes. Software that tracks browsing activity to show targeted ads.
Adware Malware that displays unwanted advertisements to the user. Popup ads that appear constantly while browsing the web.
Rootkit A tool that hides the existence of certain processes or programs from normal detection methods. Hacker installing a rootkit to avoid detection by antivirus software.
Keylogger Malware that records keystrokes to capture passwords, credit card numbers, and other sensitive information. Software that secretly logs everything typed on a computer.
Botnet A network of infected devices controlled by a hacker, often used for distributed denial-of-service (DDoS) attacks. A network of zombie computers used to flood a website with traffic.
Backdoor A secret method for bypassing normal authentication or security measures to gain remote access to a system. Malware that installs a backdoor for future access to an infected computer.
Fileless Malware Malware that runs in the system memory and does not leave traditional traces in the file system. Malware that executes directly from the computer's memory, avoiding detection by traditional antivirus programs.
